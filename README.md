# Ebay - backend app

#### Simple backend Ebay clone. As a user, you can register, log in, add an item to sell, view items to sell.


### Tech stack used: ###
- Java SE
- IntelliJ IDEA
- Gradle
- MySQL
- H2
- Spring Boot
   - Spring Web
   - Spring Developer Tools
   - Spring Data JPA
   - Hibernate
   - Spring Security
   - Lombok
- JSON
